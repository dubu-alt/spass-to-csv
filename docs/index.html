SPass CSV 변환기

Samsung Pass 내보내기(.spass) 파일을 Chrome/Proton CSV 또는 Bitwarden JSON으로 변환

설치가 필요 없습니다. 모든 복호화·변환은 이 브라우저 안에서만 실행되며, 파일과 비밀번호는 어떤 서버에도 전송되지 않습니다. 인터넷을 끊고 사용해도 됩니다.

여기에 spass_export_data.spass 파일을 끌어다 놓거나 클릭해서 선택

출력 파일에는 평문 비밀번호가 들어 있습니다. 다른 앱으로 가져오기가 끝나면 반드시 파일을 안전하게 삭제하세요.

지원 형식 / 동작 방식

Base64(salt 20B + IV 16B + AES-256-CBC 암호문) 구조의 Samsung Pass 내보내기를 PBKDF2-HMAC-SHA256(70,000회)로 키를 만들어 복호화한 뒤, semicolon 테이블을 파싱합니다. 비밀번호 우회 기능은 없습니다.
